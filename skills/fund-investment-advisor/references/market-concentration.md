# 市场集中度分析方法

## 概念

市场集中度衡量一个行业中头部企业占据市场份额的高低，用于判断市场是少数巨头垄断还是充分竞争。

## 核心指标

### CRn（Concentration Ratio，集中度比率）

CRn = 前n家企业市值之和 / 行业总市值 × 100%

| 指标 | 含义 | 参考值 |
|------|------|--------|
| CR4 | 前4名占比 | >60% 高集中，30-60% 中等，<30% 低集中 |
| CR8 | 前8名占比 | >70% 高集中，40-70% 中等，<40% 低集中 |
| CR10 | 前10名占比 | 常用于A股分析 |
| CR20 | 前20名占比 | 大盘蓝筹影响力指标 |
| CR50 | 前50名占比 | 整体市场结构指标 |

### HHI（赫芬达尔-赫希曼指数）

HHI = Σ(市占率_i²)，其中市占率以百分比表示

| HHI值 | 市场结构 |
|-------|---------|
| < 1000 | 竞争充分 |
| 1000-1800 | 适中集中 |
| > 1800 | 高度集中 |

**计算示例：**
```python
# 前100名公司的HHI
market_shares = top100['总市值'] / total_market_cap * 100
hhi = (market_shares ** 2).sum()
```

## A股市场集中度（2025年6月数据）

- A股总市值：约100万亿元
- CR4：约8%（茅台、移动、工行、建行）
- CR8：约13%
- CR20：约21%
- HHI：约48（竞争充分）

**结论：** A股市场集中度较低，竞争充分，没有形成高度垄断。

## 使用AKShare获取数据

```python
import akshare as ak

# 获取A股实时行情
df = ak.stock_zh_a_spot_em()

# 按市值排序
df_sorted = df.sort_values('总市值', ascending=False)

# 计算CRn
total_cap = df['总市值'].sum()
for n in [4, 8, 10, 20, 50]:
    top_n_cap = df_sorted.head(n)['总市值'].sum()
    cr_n = top_n_cap / total_cap * 100
    print(f"CR{n}: {cr_n:.2f}%")

# 计算HHI
top100 = df_sorted.head(100)
shares = top100['总市值'] / total_cap * 100
hhi = (shares ** 2).sum()
print(f"HHI: {hhi:.0f}")
```

**注意：** 需要配置代理才能访问东方财富API（端口7897）。

## 投资启示

1. **低集中度市场** → 行业轮动和政策导向更重要
2. **高集中度行业** → 龙头股效应明显，关注行业前三
3. **HHI变化趋势** → 集中度提升说明行业整合加速
