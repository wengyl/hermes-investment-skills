# 产业链拓扑详细文档

## 概述

本文档详细描述 9 条主要产业链的拓扑结构、传导逻辑和投资含义。

---

## 1. 半导体产业链

### 拓扑图
```
EDA/IP ─────→ IC设计 ─────→ 晶圆制造 ─────→ 封装测试
  │              │              ↑                ↑
  │              │              │                │
  └──────────────┼──────────────┼────────────────┘
                 │              │
            半导体设备 ──→ (产能扩张)
            半导体材料 ──→ (成本传导)
```

### 传导逻辑
| 事件 | 传导路径 | 投资含义 |
|------|----------|----------|
| EDA工具涨价 | EDA→IC设计(成本+1-2月) | 设计公司毛利率承压 |
| 晶圆厂扩产 | 设备→晶圆(产能+6月)→封测(需求+1月) | 设备先行，封测滞后受益 |
| AI芯片需求增 | IC设计→晶圆(需求+2月)→封测(需求+1月) | 全链受益，设计端弹性最大 |
| 材料供给紧张 | 材料→晶圆(成本+1月)→封测(成本+1月) | 制造端成本压力最大 |

### 关键监控指标
- 全球半导体销售额 (WSTS)
- 晶圆厂产能利用率
- 存储芯片价格 (DRAM/NAND)
- 设备订单 (SEMI Book-to-Bill)

## 2. AI算力产业链

### 拓扑图
```
数据要素 ──→ 大模型 ──→ AI应用
               ↑
AI芯片 ──→ 算力基建
```

### 传导逻辑
| 事件 | 传导路径 | 投资含义 |
|------|----------|----------|
| GPU供给紧张 | AI芯片→算力基建(成本+1月) | 算力基建成本上升，但需求刚性 |
| 大模型迭代加速 | 大模型→AI应用(技术替代+2月) | 应用端估值重估 |
| 数据要素政策 | 数据→大模型(成本+2月)、数据→AI应用(需求+3月) | 数据资产价值重估 |

### 关键监控指标
- GPU交货周期
- 数据中心建设进度
- 大模型Benchmark排名
- AI应用MAU/DAU

## 4. 新能源车产业链

### 拓扑图
```
锂矿 ──→ 电池材料 ──→ 动力电池 ──→ 整车 ──→ 充电桩
                                  │
                                  └──→ 电池回收 ──→ 电池材料(循环)
```

### 传导逻辑
| 事件 | 传导路径 | 投资含义 |
|------|----------|----------|
| 锂价下跌 | 锂矿→材料(-1月)→电池(-1月)→整车(-2月) | 全链成本下降，整车弹性最大 |
| 电池技术突破 | 电池→整车(成本+2月) | 整车竞争力提升 |
| 充电桩政策 | 充电桩→整车(需求+3月) | 消除里程焦虑，促进销量 |

### 关键监控指标
- 碳酸锂价格
- 电池产能利用率
- 新能源车月度销量
- 充电桩保有量

## 5. 光伏产业链

### 拓扑图
```
硅料 ──→ 硅片 ──→ 电池片 ──→ 组件 ──→ 电站
                                      ↑
                     电站需求反馈 ←────┘
```

### 传导逻辑
| 事件 | 传导路径 | 投资含义 |
|------|----------|----------|
| 硅料价格下跌 | 硅料→硅片→电池片→组件(成本逐级传导) | 组件端成本下降，利润空间扩大 |
| 装机量超预期 | 电站→组件(需求+2月)→电池片→硅片 | 全链受益，上游弹性大 |
| 技术路线切换 | 电池片(技术替代) | 旧路线产能减值，新路线溢价 |

## 9. 农业产业链

### 拓扑图
```
种植 ──→ 饲料 ──→ 养殖 ──→ 屠宰加工 ──→ 食品加工
  │                                      ↑
  └──────────────────────────────────────┘
```

### 传导逻辑
| 事件 | 传导路径 | 投资含义 |
|------|----------|----------|
| 猪价上涨 | 养殖→屠宰(成本+1月)→食品(成本+1月) | 养殖端利润增厚，下游成本承压 |
| 饲料涨价 | 饲料→养殖(成本+1月) | 养殖成本上升，利润压缩 |
| 粮食安全政策 | 种植→饲料(成本+2月) | 种业受益，饲料成本变动 |

### 猪周期传导
```
猪价下跌 → 养殖亏损 → 产能去化 → 供给减少 → 猪价上涨 → 养殖盈利 → 产能扩张 → 供给增加 → 猪价下跌
周期长度: 约3-4年
```

## 传导延迟参考

| 传导类型 | 典型延迟 | 说明 |
|----------|----------|------|
| 成本推动 (cost_push) | 1-2月 | 价格信号传导较快 |
| 需求拉动 (demand_pull) | 2-3月 | 计划→采购→生产周期 |
| 产能联动 (capacity_link) | 3-6月 | 建设周期较长 |
| 技术替代 (tech_substitute) | 6-12月 | 技术扩散需要时间 |
| 政策传导 (policy_transmit) | 1-3月 | 取决于政策力度 |
